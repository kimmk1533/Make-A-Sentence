Make A Sentence 기획

뱀서류 게임
 - 자동 공격
 - 플레이어는 이동과 조합에만 신경 씀
 - 무한 맵

조작키
 - 이동: Wasd
 - 인벤토리: Tab
 - 회피(텔레포트): Space Bar

플레이어 스탯
 - 체력
 - 이동 속도
 - 주변 범위 (단어 중 주변 적을 판별하는 범위)

게임 특징 및 규칙
 - 레벨 업 시 { 단어, 단어 }의 선택지 중 원하는 것 하나를 선택해서 획득함
 - 인벤토리에서 획득한 단어를 조합해서 문장을 만듦
 - 인벤토리가 열려 있을 땐 게임 정지
 - 만든 문장은 그 즉시 시스템이 되어 효력이 발생함
 - 기본적으로 이동과 인벤토리를 제외한 모든 행동은 마법임 (공격, 힐, 텔레포트 등)
 - 문장으로 생성된 마법은 해당 문장의 주어에서 발동됨
 - 밸런싱 아이디어 (단어와 문장에 가중치를 넣어서 일정 가중치 이하에서만 발동 가능 하도록, 좋은 단어 or 좋은 문장일수록 높은 가중치)

추가 아이디어
 - 속성 아이디어 (예시: 포켓몬의 속성 or baba is you에서 melt나 float)
 - 하드모드 아이디어 (플레이어가 수정할 수 없는 적이 마법을 사용하는 랜덤한 문장이 추가됨.)
 - 문장에서 주어에 단어를 넣는걸 빼고, 객체마다 문장을 가질 수 있게 바꿀까? 인벤토리에서 객체에게 문장을 드래그해서 주는 방식으로
 - 마법을 플레이어 마법, 적 마법으로 구분하기? 플레이어 마법은 모든 적에게 적용되서 ㅁ마법을 사용하는 문장에 사용 가능하고, 적 마법은 그냥 기믹으로만 사용?

마법의 종류
 - 투사체 마법: 목표를 향해 날아가는 마법 (예시: 파이어 볼, 라이트닝 볼트, 도끼 투척 등)
 - 즉발 마법: 시전 즉시 목표에게 적용되는 마법 (예시: 힐, 쉴드, 텔레포트 등)
 - 버프 마법: 시전 후 일정 기간 동안 적용되는 마법 (예시: 재생, 헤이스트, 슬로우, 블랙홀 등)

구상중인 투사체 마법
 - 파이어 볼: 파이어 볼을 발사한다
 - 라이트닝 볼트: 라이트닝 볼트를 발사한다
 - 매직 미사일: 매직 미사일을 발사한다
 - 매직 미사일(유도): 목표에게 유도되는 매직 미사일을 발사한다

구상중인 즉발 마법
 - 힐: 즉시 체력을 회복한다
 - 쉴드: 일정 대미지를 보호하는 쉴드를 얻는다
 - 텔레포트: 현재 이동 방향으로 순간 이동한다

구상중인 지속 마법
 - 재생: 일정 시간 동안 체력을 회복한다
 - 거대화: 일정 시간 동안  크기가 커진다
 - 소형화: 일정 시간 동안  크기가 작아진다
 - 헤이스트: 일정 시간 동안  해당 객체의 이동 속도가 빨라진다
 - 슬로우: 일정 시간 동안  해당 객체의 이동 속도가 느려진다
 - 속박: 일정 시간 동안  해당 객체의 이동을 멈춘다
 - 블랙홀: 일정 시간 동안 주변 모든 객체를 빨아들이는 블랙홀을 생성한다
 - 안티 매직: 일정 시간 동안 플레이어의 마법가 적의 마법를 지운다
 - 땅울림: 일정 시간 동안  해당 객체를 제외한 주변 모든 객체에게 이동한 거리에 비례한 대미지를 준다

구상중인 단어
 - 플레이어
 - 적 (적의 종류만큼 있음) (출현 가능한 적만 등장)
 - 마법 (마법의 종류만큼 있음)

주어 선택 방식
 - 가장 가까운 (1)
 - 랜덤 (1~all)
 - 플레이어 주변 (all)

목적어 선택 방식
 - 가장 가까운 (1)
 - 랜덤 (1~all)
 - 주어 주변 (all)

더 생각해봐야할 문장
 - {플레이어 / 개별 적 / 모든 마법 / 개별 마법}이/가 ㅁ마법을 사용할 때, (가장 가까운 / 모든) {플레이어 / 주변 적 / 개별 적}에게 ㅁ마법을 사용합니다.
=> 이건 좀 더 생각해봐야할 듯 (해당 문장에 대한 마법 사용 그래프를 만들어서 사이클이 존재하는 지 판별하면 될 수도?)
 - {모든 즉발 마법 / 모든 지속 마법 / 개별 즉발 마법 / 개별 지속 마법}이/가 끝날 때 마법의 타겟이 (가장 가까운 / 모든) {플레이어 / 주변 적 / 개별 적}에게 ㅁ마법을 사용합니다
=> 얘도 사이클 판별해야 될 듯?

현재 문제
 - 문장은 어떻게 획득함? => 뱀서 상자처럼? => 그럼 획득량이 너무 적지 않음? => 레벨 업 했을 때 { 문장 / 단어 / 지속 효과 } 중에 고르게 할까?
 - 에셋은 어떻게 함? => 일단 프로토타입을 만들어서 재밌는 지 부터 보자
 - 만들 수 있는 문장 종류가 너무 적은거 아님? => 일단 프로토타입을 만들어서 주변인에게 조언을 구하자
